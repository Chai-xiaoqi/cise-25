instructions on how to use this react app

1. pull this repository to your local repository, enter command: git remote add origin master:master(or any local branch you like) https://github.com/Chai-xiaoqi/cise-25.git
2. open two terminals: one is for client, the other one is for server.
    2.1. on the client side, enter the command: npm install, then enter the command: npm start
    2.2 on the server side, enter the command: npm install, then enter the command: npm run app
then, you can see the app is running on the http://localhost:3000/
